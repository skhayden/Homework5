This research is designed to help us understand what the most popular names are. Specially the top ten female names. 
#############################################
#File and object description 
Data files used in research
yob2015- table of names by gender and how many times they were used in 2015.
yob2016- table of names by gender and how many times they were used in 2016.

Objects
Df –raw data from the yob2016 file 
Y2016- tidy data from the yob2016 file
Y2015 – raw data from the yob2015 file
Final- tidy data of both years merged together. An additional column was added from the total vector. 
Total- the sum of the count of each name between year 2015 and 2016.
Girls- a list of top ten girl’s names
